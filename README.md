# PricePerSize

Compare product prices by unit to find the best value. Works for groceries, household items, and more.

## Features

- **Price Calculator**: Compare 2+ products by price per unit
- **Quick Tag Check**: Verify if shelf tag math is correct
- **Multi-currency support**: USD, EUR, GBP, INR, and more
- **Unit conversion**: Automatically converts between g, kg, oz, lb, ml, L, etc.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Frontend**: React 18
- **Styling**: CSS with custom design system
- **Deployment**: Vercel

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Deploy to Vercel

1. Push to GitHub
2. Import project in Vercel
3. Add custom domain: `pricepersize.site`
